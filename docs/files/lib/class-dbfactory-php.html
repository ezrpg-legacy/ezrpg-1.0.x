public static function factory($type = 'mysql',

$username = 'root',
$password = '',
$db = 'ezrpg')

A static function to return the correct database object according to the database type.

Parameters

$typeThe type of database driver to use.
$hostThe host of the database server.
$usernameUsername to login with.
$passwordPassword to login with.
$dbName of the database.

Returns

A new instance of the database driver class.

Throws a DbException on failure.

Example Usage

try
{
    $db = DbFactory::factory('mysql', 'localhost', 'root', 'password', 'ezrpg');
}
catch (DbException $e)
{
    $e->__toString();
}
