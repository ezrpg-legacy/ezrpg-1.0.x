factory

A factory class for creating module objects.  Includes the file /module/$module/index.php then creates a new instance of the class Module_$module.

Example: index.php?module=Index

Looks for /modules/Index/index.php, then creates new Module_Index() object.

Parameters

$dbAn instance of the database object.
$tplAn instance of the smarty object.
$moduleName of the module, defaults to Index.

Returns

A new instance of a module class.

Shows the Index module if the specified module cannot be found.

Example Usage

$new_module = ModuleFactory::factory($db, $tpl, $player);
$new_module->start();
